# 상관분석
 - 목표: **연속형** 변수 X, Y가 **선형적인** 상관관계를 가지는가?
 - [[3. Resource/통계학개론/피어슨 상관계수\|피어슨 상관계수]] $\rho$ 로 표현. 
	 - [[3. Resource/통계학개론/피어슨 상관계수#^db3453\|가정]] 만족 필요
	 - $\rho$ = $\frac{Cov(X,Y)}{\sqrt{Var(X)*Var(Y)}}$
		 - $Cor(X,Y)$  = X, Y의 변화의 방향관계인데, 표준편차의 곱으로 나누어줘 단위를 없애준 것
		 - -1<=_$\rho$<=1
 - 가설검정: H0 : ρ= 0 , H1 : ρ̸= 0 -> 기각될 경우, $\rho$ 는 0은 아니어서 상관관계는 있음
	 - 검정통계량 = $t = r\sqrt{\frac{n-2}{1-r^2}}$ ~ $t(df = n-2)$ 
 - 비모수적 방법: [[3. Resource/통계학개론/스피어만 상관계수\|스피어만 상관계수]]

# 회귀분석이란

- 변수간의 함수적 관련성을 구하기 위해
- 모형을 가정하고
- 그 모형을 측정된 변수의 데이터로부터 추정하는 통계적 방법

최소제곱법에서 다시 돌아와서, 
최소제곱법이 하려는 것은 식을 가장 잘 설명하는 $\beta_1$ 을 찾는 것인데, 샘플링했기 때문에 이 역시 분포를 가진다. 
이 분포를 통해 $\beta_1$ 을 추정하기 위한 가설검정을 해보자 
▶ 가설H0 : β1 = 0, H1 : β1 ̸= 0
여기서, 직접적으로 β1의 검정통계량을 통해 (상관계수 가설검정과 정확히 식이 동일함) 구할 수도 있으나, 
아노바 테이블을 통한 F 통계량 검정으로도 구할 수 있다. (둘은 서로 제곱하면 같은 관계기 때문)

잔차의 합은 다음과 같이 쓰여진다
$\sum(Y_i - \bar{Y})^2$(SST, 총제곱합)  = $(n-1)S_Y^2$  -> 분산의 합(n-1은 자유도 때문)
= $\sum(Y_i - \hat{Y_i})^2 + \sum(\hat{Y_i} - \bar{Y})^2$ = SSE + SSR -> 오차 중 잔차가 설명하는 부분+
= $(n-1)(S_Y^2 - \beta_1^2 S_X^2)$ + $\beta_1^2 (n-1)S_X^2$ -> n-1 묶어내고 나면, 전체 분산 - X직선으로 설명 가능한 부분,

가슴이 웅장해진다
이렇게 총제곱합을 분산 간의 합으로 정의한 뒤, 아노바 테이블을 적용하면

![스크린샷 2026-03-21 오후 4.55.43.png](/img/user/5.%20Attatchments/%EC%8A%A4%ED%81%AC%EB%A6%B0%EC%83%B7%202026-03-21%20%EC%98%A4%ED%9B%84%204.55.43.png)
이렇게 되는데, 여기서 F 통계량을 구할 수 있다. 
만약, 이 F통계량에 대한 p값이 유의수준보다 작다면, 귀무가설을 기각하고 
H1 : β1 ̸= 0이라는 가설을 채택하게 되고, 그때 b1 값이 회귀 기울기로 추정할 수 있다. 

또한, [[3. Resource/통계학개론/결정계수 (R-squared)\|결정계수 (R-squared)]] $R^2 = \frac{SSR}{SST}$ 인데, Y의 총변동(SST) 중 X에 의해 설명되는 (SSR) 의 비율 을 나타낸다![스크린샷 2026-03-21 오후 5.02.35.png\|해석 예시](/img/user/5.%20Attatchments/%EC%8A%A4%ED%81%AC%EB%A6%B0%EC%83%B7%202026-03-21%20%EC%98%A4%ED%9B%84%205.02.35.png)
